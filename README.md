# Student-Database-Management-System-using-Singly-Linked-List---Data-Structure-Algorithm
A C-based menu-driven Student Record Management System using Singly Linked List. Supports 15 operations: insert at begin/end/middle, delete, search by name, sort by roll number, recursive &amp; iterative traversal, reverse print, in-place reversal, and file save/load using std.txt.
